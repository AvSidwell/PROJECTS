# Fashion Collection

Luxury fashion portfolio — Next.js 16 + TypeScript + Tailwind CSS

## Stack
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS 4
- Local fonts: Panchang, Melodrama, Gambetta

## Dev

```bash
npm run dev
```

## Content

All content is driven from `src/data/products.json`.
Images live in `public/images/`.
